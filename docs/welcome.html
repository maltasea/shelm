Welcome to Buoy

A small language with a practical compiler pipeline and multi-target code generation.

Source: .by Targets: perl / ocaml / go / bytecode keyword ... end blocks

1) First Program

let name = "World"
println("Hello, " ++ name ++ "!")
opam exec -- ./_build/default/bin/buoy.exe hello.by --target perl > hello.pl
perl hello.pl

2) Build and Test

opam exec -- dune build
opam exec -- dune runtest
If opam is already active in your shell, you can omit opam exec --.

3) Core Syntax

  • let x = 1, x = x + 1
  • if cond then ... else ... end
  • while cond do ... end
  • for item in items do ... end
  • fn add x, y { return x + y }
  • rec fn fact n { ... }
  • No implicit lambdas/closures in the speed-first profile

4) FFI Host Syntax

  • Read: $foo/bar
  • Call: &foo/bar(1, 2)
BUOY_PERL_HOST=runtime/perl_host_default.pl perl hello.pl

5) Benchmark Mode

opam exec -- ./_build/default/bin/buoy.exe benchmarks/prime_count.by \
  --benchmark benchmarks/prime_count.pl \
  --iterations 10

6) Conformance Check

./scripts/check-conformance.sh

Runs build/tests plus compile/runtime checks against the implemented spec.
