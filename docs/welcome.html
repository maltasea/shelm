Shelm Docs Site

Fast Language, Explicit Costs

Shelm is a speed-first language: bytecode-oriented, deterministic syntax, and explicit runtime tradeoffs. This page is fully local and openable directly as docs/welcome.html.

At a Glance

  • Source extension: .by
  • Targets: perl, ocaml, go, bytecode
  • Canonical style: keyword headers + end

Language Priorities

  • Translate to S-exprs by syntax form only
  • No significant whitespace
  • No implicit closures in core profile

Language Section

Core Forms

type age = int
enum color do
  red
  green
  blue
end

let x = 1
x = x + 1

if x > 0 then
  println("ok")
else
  println("no")
end

while x < 10 do
  x = x + 1
end

foreach item in [1, 2, 3] do
  println(string_of(item))
end

Block syntax is keyword/end only (`then`, `with`, `do` ... `end`).

Examples

Match With Cases

let total = 6
match total with
  | 0 -> println("zero")
  | 6 -> println("six")
  | _ -> println("other")
end

Loop Control

let i = 0
while i < 10 do
  i = i + 1
  if i == 3 then
    continue
  end
  if i == 7 then
    break
  end
end

Host FFI

let ms = $time/now_ms
let sum = &math/add(20, 22)
println(string_of(ms))
println(string_of(sum))

Form-Only S-Expr Lowering

Shelm is designed so syntax lowers deterministically to S-expr-like canonical forms without semantic analysis.

Surface

if x > 0 then
  y = y + 1
else
  y = 0
end

Shape

(if (> x 0)
  (do (set y (+ y 1)))
  (do (set y 0)))

Performance Profile

  • Bytecode target is optimization anchor.
  • Syntax favors deterministic parsing and cheap lowering.
  • No implicit closure/lambda syntax in the core profile.
  • Runtime-costly features should be explicit and opt-in.
